# RF76: Empleado Lee Balance

**Última actualización:** 01 de marzo de 2025

### Historia de Usuario

Como empleado, quiero consultar mi saldo actual de puntos y detalles de mi cuota asignada, para planificar mis compras en función de mis recursos disponibles.

### Criterios de Aceptación:

- El sistema debe permitir a los Empleados leer el balance de un cliente.
- El balance debe incluir información sobre el total de compras, pagos realizados y saldo pendiente.
