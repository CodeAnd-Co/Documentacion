**Última actualización:** 08 de mayo de 2025

### Historia de Usuario

  **Criterios de Aceptación:**
  - El sistema debe de mandar un correo de cambio de contraseña
  - El sistema debe de aceptar la nueva contraseña despues de cambiarla
