# CL1 Checklist de Procesos

### Objetivo

Definir los criterios que deben cumplis los productos de trabajo que formaran parte de una linea base

### Criterios de aceptacion

- [ ] El procducto de trabajo se encuentra registrado en la matriz de configuracion con su ultima version.
- [ ] El producto de trabajo debe estar en la rama `main` del repositorio de github de la wiki
- [ ] El producto de trabajo debe estar auditado
